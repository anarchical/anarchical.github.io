# 笔记博客记事本

### 博客模板
https://vuepress-theme-hope.github.io/v2/

### init project
```shell
pnpm install
```

### run project
```shell
pnpm docs:dev
```

### clean run project
```shell
pnpm docs:clean-dev
```

### build project
```shell
pnpm docs:build
```

### update project
```shell
pnpm add vuepress@next vuepress-theme-hope@next && pnpm i && pnpm up
```
